📌 API 사용법

이 API는 전 세계 카페 목록을 관리합니다.

  • [GET] /cafes - 모든 카페 목록 조회
  • [GET] /cafes/random - 랜덤 카페 추천
  • [GET] /cafes/location/{location} - 특정 지역 카페 검색
  • [POST] /cafes - 새로운 카페 추가
  • [POST] /cafes/{cafe_id}/update-request - 카페 정보 수정 요청
  • [DELETE] /cafes/{cafe_id} - 카페 삭제 (관리자 전용)

자세한 내용은 API 문서를 확인하세요.
